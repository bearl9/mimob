MiMo Chat
All features · tap to learn
💬 Conversations
New Tap + in the sidebar header to start a new chat.
Rename Tap ✏️ on any conversation to rename it inline.
Delete Tap 🗑️ — moves to Recently Deleted, not gone yet.
Restore Scroll to Recently Deleted at the bottom → tap ↩️.
Auto-purge Trashed chats are permanently deleted after 7 days.
📎 Attachments
🖼️ Image Tap ＋ to attach an image, or paste / drag & drop on desktop.
📄 PDF Extracts text from up to 30 pages automatically.
📝 File Attach code or text files (.py, .js, .html, .json, etc.).
🎵 Audio Attach audio files — MiMo acknowledges them in context.
🌐 Browse Paste any URL to fetch and attach webpage content as context.
🎤 Voice Tap the mic to dictate — transcribed into the message box.
🔖 Bookmarks & Pinning
Full Message Tap 📌 on any MiMo reply to pin the entire message — always included in future API calls.
Block Snippet Tap the 🔖 that appears beside any paragraph or code block to pin just that block.
Selection Highlight text → tap "🔖 Bookmark Selection" to pin an exact snippet.
Always included Pinned content is injected into every API call regardless of conversation length.
🗂️ Context Management
Window Only the last 20 messages are sent per request — older ones are silently dropped.
Bookmarks Pinned messages are always included, no matter how old.
Clear Context Tap Clear context in the sidebar footer to reset without deleting the conversation.
📓 Diary
Track Switch to the Diary tab in the sidebar to log Food, Sleep, Water, Exercise, Medication, Weight, Symptoms, Expenses, or Notes.
AI Log Tap any category to open it in the main chat area — log entries conversationally.
History Each category shows a running history of past entries.
☁️ Cloud Sync
Optional Tap Sign in to sync in the sidebar footer to enable cross-device sync.
How it works Your chats are stored in Firebase (Google). Last save wins across devices.
Privacy See the 🔒 Policy tab for full data disclosure before signing in.
💡 Tips
Send Tap Send or press Enter. Use Shift+Enter for a new line on desktop.
Copy code Every code block has a Copy button in the top-right corner.
Copy message Tap 📋 on any message to copy the full text.
Regenerate Tap 🔄 on any MiMo reply to regenerate the response.
Auto-retry Empty API responses are retried up to 3 times automatically.
TTS Tap 🔊 on any reply to hear it read aloud.
Privacy & Usage Policy
🔒 Privacy Policy
Effective: 17 May 2026 · Governed by Singapore PDPA 2012
What we collect
  • Chat messages & conversation history — stored locally on your device
  • Diary entries (food, sleep, water, etc.) — stored locally only
  • Google name, email & profile photo — only if you sign in for sync
  • App settings (API key, voice config) — stored locally, never transmitted
How we use it
  • To run MiMo Chat and generate AI responses via the MiMo API
  • To sync your data across devices when you choose to sign in
  • We do not sell, share, or use your data for advertising
Third-party services
  • Google Firebase — authentication & cloud sync (only when signed in)
  • MiMo API — your messages are sent here to generate responses
  • Jina AI / AllOrigins — used only when you use the Browse feature
⚠ If you sign in, your chat data is stored in Google Firebase. The developer (Bearl) has administrative access to this database. Do not store highly sensitive personal information in chats.
Your rights (PDPA)
You have the right to access, correct, and withdraw consent for your personal data. To request data deletion, sign out and contact bear.leongzq@gmail.com.
Data retention
Local data: until you clear your browser data. Cloud data: until you request deletion via email.
